# Updin: Plataforma de Educação Financeira para Adolescentes

O Updin é uma plataforma mobile de educação financeira para adolescentes, com gamificação, gestão de mesada, missões, quizzes, ranking e acompanhamento parental.

## Repositórios

- [Frontend mobile](https://github.com/gaformario/updin-backend-api)
- [Backend API](https://github.com/gaformario/updin-frontend-mobile)

## Tecnologias

- React Native
- Node.js
- NestJS
- PostgreSQL
- Prisma ORM
- GitHub Actions (CI/CD)
- AWS
